Where biomedical data agents fail: a trap taxonomy

We benchmarked five open biomedical AI agents — Biomni, CellVoyager, MEDEA, OmicsClaw and K-Dense — on blind data-quality questions over real biomedical datasets: CDISC clinical trials, LINCS, TCGA and single-cell atlases. We used Natural Joints to find the traps in that data and to judge, for each, whether a correct answer can be recovered or the data is simply missing. The result reduces to one line: a stronger model fixes convention errors and none of the semantic-grounding errors.

01Method

Each agent runs locally on the same CSV files and answers the same blind analyst questions; the scaffold is the only variable. Four agents share a Gemini backbone (gemini-2.5-pro, retested on gemini-3.1-pro); K-Dense's code executor is Claude Opus 4.8, a cross-family control. On a single-cell baseline of well-known pitfalls (pseudoreplication, batch confound, QC contamination) every agent passes — the failures below are signal, not general incompetence.

02How Natural Joints built the benchmark

Natural Joints profiles raw tables and alerts on nonconventional usage: a coded field read literally, a one-to-many join that inflates a per-entity mean, a column of zeros that behaves like missing data, a foreign key with no parent table, a table absent from the delivery. We used those alerts to locate real traps in the datasets above and converted each into a blind question with a known answer.

For every candidate Natural Joints makes a second judgment: whether the correct answer is recoverable from what was delivered, or not. A masked value is recoverable — the layer carries the value map. A dropped table is not — the correct answer is "this is missing." That determinate / indeterminate split is the first axis of the taxonomy.

We verified that Natural Joints flags every trap family in the benchmark. Masked columns and values surface as low-confidence reference pointers; the fan-out and dangling-key joins raise structural alerts; the dropout-zero columns score 0.88–0.94 on its zero-as-missing detector, and the opposite on a control gene where zeros are biological; the dropped table raises a completeness alert. Each ground truth was then recomputed independently from raw data.

05Data & ground truth

  • Data. CDISC ADaM pilot (clinical trials); an anonymized LINCS L1000 screen (masked names, held-out dictionary as ground truth); TCGA colorectal expression matrices; the Human Lung Cell Atlas as the single-cell baseline.
  • Ground truth. Recomputed independently from raw data. The LINCS counts and joins are deterministic; the zero-as-missing cases are validated against the Natural Joints detector, with a biological-zero control gene. None are outlier-driven.
  • Scope. A taxonomy of failure modes, not a leaderboard: which families a bigger model closes (conventions) and which it never touches (grounding).
