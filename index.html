Where biomedical data agents fail: a trap taxonomy

We ran two open biomedical AI agents head-to-head — Biomni and CellVoyager — on blind data-quality questions over real biomedical datasets: CDISC clinical trials, LINCS and TCGA. Three more agents (MEDEA, OmicsClaw, K-Dense) could not be run across the full task; see Method. We used Natural Joints to find the traps in that data and to judge, for each, whether a correct answer can be recovered or the data is missing.

01Method

Two agents ran every family and anchor this study: Biomni (SNAP, Stanford) and CellVoyager (Zou lab, Stanford). Both execute locally on the same CSV files, answer the same blind analyst questions, and run on gemini-3.1-pro — one run each, so the numbers carry run-to-run variance. The other three agents could not be evaluated across the taxonomy and are mentioned only for context: MEDEA refuses masked or anonymized data (and declined the plain gene-expression questions too), OmicsClaw fails to run on gemini-3.1, and K-Dense's Claude executor was limited to the CDISC subset. On a single-cell baseline of well-known pitfalls (pseudoreplication, batch confound, QC contamination) both agents pass — the failures below are signal, not general incompetence.

02How Natural Joints built the benchmark

Natural Joints sourced the traps and verified, for each question, whether it can be answered from the delivered data or the data is missing.

06Data & ground truth

  • Data. CDISC ADaM pilot (clinical trials); an anonymized LINCS L1000 screen (masked names, held-out dictionary as ground truth); TCGA colorectal expression matrices; the Human Lung Cell Atlas as the single-cell baseline.
  • Ground truth. Recomputed independently from raw data. The LINCS counts and joins are deterministic; the zero-as-missing cases are validated against the Natural Joints detector, with a biological-zero control gene. None are outlier-driven.
  • Scope. A taxonomy of failure modes, not a leaderboard: which families a bigger model closes (conventions) and which it never touches (grounding).

Natural Joints — data & code auditing for biomedical analysis. We find bugs in how scientific data is used: wrong join grain, wrong analysis flags, train/test leakage, confounds.
